Encabezado Celda 1 Encabezado Celda 2 Encabezado Celda 3
Contenido Celda 1 Contenido Celda 2 Contenido Celda 3
``` CSS: ``` table { width: 100%; border-collapse: collapse; table-layout: fixed; } td, th { padding: 10px; border: 1px solid black; text-align: center; } th { background-color: lightgray; } ``` Para hacer que la tabla se expanda en vertical, lo que necesitas es no definir una altura fija para ninguna de las filas o celdas (td/th).
